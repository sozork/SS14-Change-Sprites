# Space Station 14 Sprite Changer
Инструмент для локальной смены спрайтов или скачивания спрайтов из базы данных Space Station 14(https://github.com/space-wizards/space-station-14)
Работает на серверах(Дабы добавить сервер сюда, нужно получить разрешение от хоста):
* TTMC14 - https://github.com/ttmc14/ttmc14

Руководство по пользованию:
Дабы сменить или скачать спрайт нужно:
* Нажать “Change Sprites”
* Выбрать путь до базы данных с контентом (C:\Users\user\AppData\Local\Space Station 14\launcher\content.db)
* Удалить констрейнт нажав на “remove check”
* Вписать в поле для ввода предполагаемый путь до файла в базе(Например: хотим найти бегуна - пишем “runner”)
* Найти и нажать на спрайт, который хотим сменить
* Выбрать “change” и выбрать файл на замену
* В конце по желанию можно нажать "add check", дабы вернуть констрейнт
/// пока сейвы не сделаны. Так что если вы поменяли спрайт, в он вдруг стал "плохим", то удаляйте базу данных(при заходе на сервер она скачается заново).

Как оно работает:
Space Station 14 хранит почти все спрайты (кроме тех, что запечены в движок) в базе данных, а этот инструмент их оттуда достаёт и/или заменяет.
Некоторые сервера не хотят что бы их спрайты меняли, потому их тут и нет. Я могу добавить их в список разрешённых если получу разрешение.
